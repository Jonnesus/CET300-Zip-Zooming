# CET300-Zip-Zooming
 
Arcade racing game with an adaptive difficulty system

---
## Controls
### Keyboard
Accelerate: W\
Brake: S\
Steer: A/D\
Handbrake: Space\
Change Camera: C\
Pause: Escape\
Menu Confirm: Enter

### Gamepad
Accelerate: Right Trgger\
Brake: Left Trigger\
Steer: Left Thumbstick\
Handbrake: East Button\
Change Camera: North Button\
Pause: West Button\
Menu Confirm: South Button
