# CET300-Zip-Zooming
 
Arcade racing game with an adaptive difficulty system

---
## Controls
### Keyboard
Accelerate: W
Brake: S
Steer: A/D
Handbrake: Space
Pause: Escape
Menu Confirm: Enter

### Gamepad
Accelerate: Right Trgger
Brake: Left Trigger
Steer: Left Thumbstick
Handbrake: East Button
Pause: Select
Menu Confirm: South Button
